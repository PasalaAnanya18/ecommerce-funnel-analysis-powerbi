# 📊 E-commerce Funnel Conversion Analysis

## 🔍 Overview
This project analyzes user behavior across an e-commerce funnel to identify drop-offs and improve conversion rates.

---

## 🛠️ Tools Used
- Power BI
- Excel

---

## 📈 Key Insights
- ~60% drop-off between Product View and Add to Cart
- Mobile users show higher drop-off compared to Desktop
- Overall conversion rate is ~8%

---

## 💡 Recommendations
- Improve mobile UX
- Optimize product pages
- Simplify checkout process
